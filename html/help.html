Passage Form

Name: input the name of the person filling the form.
Genoype: select genotype of line passaged.
BC/IB Number: input current inbreed or backcross number of the generation, adding +1 for the latest cross.
Hatch Date: hatch date of the line passaged.
Passage Date: input today's date.
Mating Line: the the passage is crossing with.

Sorting Form

Name: input the name of the person filling the form.
Date Sorting: input today's date.
Marker Location: input location of fluorescent marker.
Ratio of fluorescent to non-fluorescent larvae: approximate ratio using slider, with color representing fluorescent mosquitos.
Number of fluorescent animals sorted: input total number of animals which were fluorescent.
Additional Notes: open field for any relevant notes, such as male to female ratios.

Clutch Form

Name: input the name of the person filling the form.
Genoype: select genotype of line passaged.
Hatch Date: input hatch date of the clutch.
Collection Date: input date the eggs are collected.
Clutch number: indicate the first, second, or third clutch laid by the hatched mosquitos.
Egg Papers: input density of the eggs on the egg paper. See images below for reference.

Add New User

Complete the form to add a new lab user to the databsase.

After entering the administrator password, select the table to edit from the dropdown menu. Enter the sort, passage, or clutch ID of the row you want to edit in the first field, and allow the form to autopopulate with the current values in the table. The form allows users to freeform edit any field in the row. After editing the desired values, click submit. A confirmation message will appear on screen if changes were successful.
